
##Description
Simple AngularJS app to add books to a single Library

###To run locally

####clone repo
	git@github.com:tbuchanan/angular_bk_lab.git

####bundle
	Run bundle update

####create local postgres database
	rake db:create

####start local server:

	rails s
	view app at http://localhost:3000/
	
	
###gems

	gem 'rails', '4.1.1'
	gem 'pg'
	gem 'coffee-rails', '~> 4.0.0'
	
###languages

	CoffeeScript
	HTML
	Ruby
	
###framework

	AngularJS
